What is Comparative Judgement?

Comparative judgement is.......

How does it work?

It is a pair wise comparison tool. It aims to provide the user with two options and they then select what one they like the most. The process then ranks the pieces of work that was being compared.

Why are we doing this?

To create the foundations to a bigger app that will be able to mark students work from schools and training providers.
